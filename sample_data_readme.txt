# 示例销售数据说明

这个CSV文件包含模拟的销售数据，可用于测试PandasAI代码生成功能。

## 数据字段说明

- 日期: 销售日期
- 产品ID: 产品唯一标识符
- 产品名称: 产品名称
- 类别: 产品所属类别
- 单价: 产品单价（人民币）
- 销量: 产品销售数量
- 地区: 销售区域

## 可能的查询示例

1. "找出销量最高的产品类别并计算各类别的平均销量"
2. "分析不同地区的销售总额，并按销售总额降序排列"
3. "计算每个产品类别的销售总额，并用条形图展示"
4. "找出单价高于2000的产品，并分析它们在各地区的销量分布"
5. "计算每日的销售总额，并展示销售额随时间的变化趋势"
